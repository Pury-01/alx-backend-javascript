# TypeScript Project

## Key Features
* Basic types of TypeScript
* Interfaces, Classes, Functions
* Working with the DOM in TypeScript
* Generic types and namespaces

## Requirements
+ Editor **vi, vim, emacs, Vs Code**
+ Tested on Ubuntu 18.04 using __jest__(v24.9.)
+ TypeScript files should use **.ts** extension

## Setup  Instructions

1. Install dependencies:
`npm install`

2. Start Development server:
`npm run start-dev`

3. Build for production
`npm run build`

## Running Tests
Excecute tests using:
`npm run test`
